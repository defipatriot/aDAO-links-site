Catalog Edit

Add or edit entries in the curated catalog. Pick which file to edit, fill in the form, preview the merged JSON, then copy or download to commit to GitHub.

How this works

  1. Pick a file tab below (wallets, contracts, etc.)
  2. The current file's contents load automatically — you'll see existing entries
  3. Fill in the form to add a new entry, OR click an existing entry to edit it
  4. Preview the updated JSON, then Copy or Download
  5. Paste into GitHub at defipatriot/tla-chain-registry/curated/[file].json and commit
  6. Next cron run picks up the change
